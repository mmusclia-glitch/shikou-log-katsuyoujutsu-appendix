# 付録C 最新ファイル取得ツール
 
書籍『AIとの会話を、捨てない。」付録C の配布ファイルです。
本文（付録C）の操作説明とあわせてご利用ください。
 
## 収録ファイル
- Start_IndexPickup_v5_0_0.bat … 起動用バッチ
- IndexPickup_v5_0_0.ps1 … 本体スクリプト
- 最新ファイル取得ツール_v5_0_0_操作説明書.docx … 使い方の説明
- 最新ファイル取得ツール_v5_0_0_仕様書.docx … 仕様の定義
 
## 想定環境
- Windows + PowerShell を前提としています。
- 本ツール v5.0.0 は公式インデックスを参照しません（INDEX=NONE 固定。入力は SOURCE と SINCE のみ）。
 
## 使い方の概要
1. Start_IndexPickup_v5_0_0.bat と IndexPickup_v5_0_0.ps1 を同じフォルダに置く。
2. Start_IndexPickup_v5_0_0.bat を実行する。
3. SOURCE（対象フォルダ）と SINCE（基準日）を指定し、最新ファイルを取得する。
詳しい手順と確認項目は、本書 付録C を参照してください。
